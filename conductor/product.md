# Initial Concept
Python, ASE, jax Numpy, scipy, HDF5. The goal is to train Machine learning interatomic potentials and use them for high performance molecular dynamics.

# Product Definition

## Vision
To develop a high-performance molecular dynamics tool powered by machine learning interatomic potentials (MLIPs), leveraging the speed and scalability of JAX and GPU acceleration.

## Target Users
- **Scientific Researchers:** Who need accurate and efficient simulations of complex molecular systems.
- **ML Engineers / Data Scientists:** Who develop and train interatomic potentials using machine learning frameworks.

## Core Features
- **ML Potential Training:** Training interatomic potentials to predict energy and forces based on atomic configurations.
- **High-Performance MD:** Running large-scale molecular dynamics simulations with optimized performance and scalability.
- **JAX-Powered Performance:** Utilizing JAX for efficient computation and multi-GPU acceleration.

## Primary Goal
To achieve maximum performance and scalability for molecular dynamics simulations, prioritizing speed and throughput for large-scale systems while maintaining high accuracy through machine learning models.

## Platform Support
- **GPU Acceleration (JAX):** Optimized for single or multi-GPU environments to maximize computational efficiency.
